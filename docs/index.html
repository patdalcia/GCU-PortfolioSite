Organized

A Project portfolio for CST-452

By Patrick Garcia

Project Overview

Organized is an organizational tool made for students, by a student.

Inspired by online University Portals, Organized aims to provide students with a central dashboard where their Course information can be easily viewed and organized.

Course and Assignment detials at a glance. Automatically groups assignments that are due soon. All displayed in a simple and easy to use Interface that streamlines the organizational process.

Background

This project was made as a Grand Canyon University(GCU) senior project. The author of this application, and this portfolio site, is Patrick Vidal Garcia. The creation of the Organized app, and its necessary documentation, were created over the span of two courses: CST-451, and CST-452. This portfolio site is used to display the final application state, and provide details into its design, and development. To reference any documentation for this project, please refer to the documentation folder of the github repo.
